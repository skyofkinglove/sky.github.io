shiro反序列化

查阅相关资料可以知道，该漏洞的利用，涉及如下几个重要的点：

  • rememberMe cookie
  • CookieRememberMeManager.java
  • Base64
  • 加密算法
  • 加密密钥硬编码
  • Java serialization
